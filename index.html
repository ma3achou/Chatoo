Notice

- Click on the friend to open a chat window.

- Open your favorite debug toolbar, and run these functions :

Chatoo.addFriends(user, urlAvatar)
  • User : the friend name that you want to add to your list.
  • urlAvatar : url of the avatar icon (This param is Optional).

ex : Chatoo.addFriends("John Doe"); 

Chatoo.updateStatus(user, newStatus)
  • User : the user that you want to update his status
  • newStatus : two values must be choose : 1 = online/ 0 = offline

ex : Chatoo.updateStatus("John Doe", 1);
